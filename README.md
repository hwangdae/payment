# DevCamp - Pront Project
토스 페이먼츠를 통한 결제를 구현하고, 쿠폰 및 포인트 이벤트 기능을 추가한 결제 프로젝트

## Tech Stack
1. React
2. TypeScript
3. React-hook-form
4. zod

## 핵심 기능
1. 토스 페이먼츠 연동<br/>
토스 페이먼츠 API를 사용하여 결제 기능을 구현합니다.
2. 쿠폰 기능 추가<br/>
사용자가 쿠폰을 입력하고 할인을 받을 수 있는 기능을 추가합니다.
3. 포인트 기능 추가<br/>
사용자가 보유한 포인트를 사용하여 결제할 수 있는 기능을 추가합니다.
